# Design Document — Prototype 015

## Concept
Memphis Milano postmodern. Bold, playful, and deliberately chaotic. Squiggly lines, polka dots, triangles, and clashing colours create a energetic, maximalist portfolio that celebrates the iconic 1980s design movement.

## Color Palette
- Pink: #ff1493 (deep pink)
- Cyan: #00bfff (deep sky blue)
- Yellow: #ffd700 (gold)
- Green: #32cd32 (lime green)
- Background: #faf0e6 (warm off-white)
- Text: #1a1a1a

## Typography
- Headings: 'Fredoka One' sans-serif (Google Fonts) — rounded, playful
- Body: 'Nunito' sans-serif (Google Fonts)
- Hero: clamp(36px, 7vw, 64px), Fredoka One
- Body: 15px, regular weight

## Layout
- Zigzag/wavy borders via CSS clip-path and border-radius
- Polka dot background patterns
- Asymmetric, dynamic grid
- Triangles and geometric shapes as decoration
- Color blocks and stripes
- Bold, contrasting section backgrounds

## Key Interactions
- Bounce animation on hover
- Wavy underline effects
- Colorful transitions
- Standard lightbox for project images

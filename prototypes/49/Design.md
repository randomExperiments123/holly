# Design Decisions — Prototype 049

## Concept
Sweet, soft, and utterly delightful — every element is designed to feel light, airy, and candy-like. Pastel colors, rounded corners, and soft shadows create a gentle, welcoming experience.

## Layout
- Centered single column with floating card effect
- Alternating pastel backgrounds for each card
- Cloud-shaped decorative elements
- Bubble-like floating animations

## Typography
Fredoka One brings a playful, rounded friendliness to headings that perfectly matches the candy theme.

## Color Palette
Five pastel colors (pink, lavender, sky, mint, yellow) rotate across cards and sections, creating a varied but harmonious palette.

## Visual Effects
- Soft, diffuse box-shadows
- Rounded corners (24px+ on everything)
- Floating/bobbing CSS animation on decorative elements
- Pastel gradient backgrounds
- Cloud and circle decorative shapes

## Lightbox
Soft pastel-toned overlay with rounded controls and candy-colored accents.

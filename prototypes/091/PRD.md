# PRD: Prototype 091 — Kaleidoscope

## Purpose
A graphic design portfolio themed around kaleidoscopic symmetry, mandala-like decorative patterns, and psychedelic-ordered geometry.

## Visual Identity
- **Concept**: Symmetrical geometric patterns, vivid colors, mandala-inspired decorative elements, psychedelic order
- **Primary palette**: Magenta (#e91e8c), cyan (#00e5ff), gold (#ffd700), deep violet (#4a00e0)
- **Secondary palette**: Turquoise, coral, electric purple
- **Typography**: Geometric sans-serif (Poppins), decorative display font for headings
- **CSS Approach**: Radial/conic gradient patterns, rotational animations, SVG geometric overlays, repeating border ornaments

## Content
Standard 5-project portfolio grid with image lightbox, hero section, about, contact. Same copy as baseline.

## Constraints
- Pure HTML + CSS + inline JS lightbox
- Mobile-first responsive
- AVIF images from `projects/projectN/`
- Accessible with ARIA, skip link, semantic HTML
- SEO meta tags + Schema.org VisualArtwork markup

# PRD: Prototype 092 — Morse/Code

## Purpose
A tech-minimalist graphic design portfolio incorporating binary and Morse code decorative elements, signal-colored accents, and a hacker/terminal aesthetic.

## Visual Identity
- **Concept**: Binary/morse decorative elements, dot-dash patterns, tech-minimalist, signal colors
- **Primary palette**: Signal green (#39ff14), amber (#ffb000), cyan (#00e5ff), on dark base (#0a0a0f)
- **Secondary palette**: Dim gray borders, terminal beige
- **Typography**: Monospace (JetBrains Mono / Fira Code), terminal-style UI
- **CSS Approach**: Dot/dash pseudo-elements, blink animations, grid patterns, scan lines, cursor blink

## Content
Standard 5-project portfolio grid with image lightbox, hero section, about, contact. Same copy as baseline.

## Constraints
- Pure HTML + CSS + inline JS lightbox
- Mobile-first responsive
- AVIF images from `projects/projectN/`
- Accessible with ARIA, skip link, semantic HTML
- SEO meta tags + Schema.org VisualArtwork markup

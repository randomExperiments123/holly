# PRD: Neon Chinese (066)

## Purpose
Showcase Holly Matthews' portfolio through a neon-lit Chinese restaurant aesthetic. Evokes the visual language of Hong Kong/Macau nightlife — glowing neon signs, red-gold-dark colour palette, decorative borders.

## Requirements
- Dark background (#0a0500) with glowing neon red (#ff1a1a), gold (#ffd700), and orange (#ff4500) accents
- CSS glow effects on borders and headings (text-shadow, box-shadow with blur)
- Chinese-inspired decorative motifs: lantern-style circles, vertical text, border frames
- Semi-transparent card panels with neon border glow
- Maintain all 5 project cards, lightbox, skip link, SEO, Schema.org, ARIA
- Mobile-first responsive layout

## Constraints
- Pure HTML + CSS, zero JS dependencies (except inline lightbox)
- AVIF images only
- Must be self-contained
- No external fonts

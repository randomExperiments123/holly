# Prototype 057 — Holographic

## Overview
A portfolio page with a holographic / iridescent aesthetic. Prismatic gradient effects, rainbow sheen, futuristic metallic feel, and light-drenched visuals.

## Requirements
- All 5 projects with slideshow
- Lightbox with integrated image viewer
- Mobile-first responsive layout
- Accessible (ARIA, skip link, semantic HTML)
- SEO meta tags + Schema.org markup
- Pure HTML + CSS, inline lightbox JS only

## Design Direction
- Iridescent gradient backgrounds shifting between rainbow hues
- Glass-like translucent elements with blur effects
- Metallic / pearlescent text effects
- Prismatic light reflections using CSS pseudo-elements
- Dark base with rainbow accents

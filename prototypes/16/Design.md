# Design Document — Prototype 016

## Concept
Scandinavian minimalism meets hygge. Warm, inviting, and functional — like a cozy Nordic interior. Every element serves a purpose; nothing is decorative for its own sake.

## Color Palette
- Background: #f7f4f0 (warm white)
- Text: #2c3e50 (navy)
- Accent: #e8dcc8 (beige)
- Light beige for cards, navy for headings

## Typography
- Headings: 'Playfair Display', serif (elegant, warm)
- Body: 'Inter', sans-serif (clean, readable)
- Generous line-height for body text
- Playfair Display for project titles

## Layout
- Centered max-width container (1100px)
- Two-column grid on desktop, single on mobile
- Fixed header with warm white background
- Card-style projects with subtle shadows
- Soft rounded corners (12px)
- Generous whitespace throughout (80-100px sections)

## Key Interactions
- Smooth scroll navigation
- Card hover: gentle lift with shadow
- Inline lightbox for project images
- Standard navigation with download link

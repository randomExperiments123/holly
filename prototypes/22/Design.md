# Design Document — Prototype 022

## Concept
Opulent luxury meets portfolio design. Deep navy-black backgrounds create a velvet-like canvas for gold accents to shine. Ornate decorative elements, elegant serif typography, and generous whitespace convey sophistication and premium quality.

## Color Palette
- Background: #1a1a2e (deep navy-black)
- Card backgrounds: #16213e (slightly lighter)
- Primary accent: #d4af37 (gold)
- Text: #e8e0d0 (warm white)
- Secondary text: #8a8070 (muted gold-brown)
- Gold gradient for headings

## Typography
- Headings: 'Playfair Display', serif (700 weight)
- Body: 'Inter', sans-serif (300, 400 weight)
- Playfair Display gives elegance and gravitas
- Inter provides clean readability for body text

## Layout
- Full dark background with subtle texture pattern
- Centered max-width container (1100px)
- Two-column grid on desktop, single on mobile
- Gold fixed header with bottom border
- Cards with gold borders and dark backgrounds
- Ornate decorative dividers (✦ separator characters)
- Border-bottom gold accents on section titles

## Key Interactions
- Subtle gold shimmer on hover
- Card lift with gold glow shadow
- Inline lightbox for project images
- Smooth scroll navigation

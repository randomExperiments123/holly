# Prototype 054 — Aztec: Design Decisions

## Color Palette
- `#8b4513` (terracotta) — primary background
- `#cd853f` (peru) — warm accent
- `#008080` (teal) — secondary accent
- `#ffd700` (gold) — highlights
- `#2f1b0e` (dark brown) — text, borders

## Typography
- Bold sans-serif (Impact / system sans, heavy weights)
- Geometric letterforms
- All-caps section headings

## Layout
- CSS stepped/zigzag borders (using repeating gradients)
- Geometric diamond and triangle motifs
- Bold, confident spacing
- Project cards: terracotta base, teal/gold accents, stepped borders

## Rationale
Mesoamerican geometric patterns provide a striking visual language. The terracotta-teal-gold palette is both historical and contemporary. Stepped motifs frame projects like temple platforms, giving the portfolio a monumental feel.

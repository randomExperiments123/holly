# Design Document — Prototype 030

## Concept
8-Bit Pixel — the portfolio is designed as an arcade cabinet interface. Everything is built with box-shadow pixel borders, chunky typography, and a retro game color palette.

## Color Palette
- Background: #2d2d2d (dark gray)
- Accent Red: #ff4136
- Accent Green: #2ecc40
- Accent Blue: #0074d9
- Text: #f0f0f0 (off-white), #aaa (muted)

## Typography
- 'Press Start 2P' (Google Fonts) — authentic pixelated retro game font
- Small font sizes due to pixel font readability
- Uppercase throughout

## Layout
- Hero with scanline overlay and pixel decorative border
- Project cards as chunky pixel "cartridges"
- Step-like pixel borders using box-shadow
- Fixed retro game-style nav bar

## Key Interactions
- Pixel hover effects (blocky color swaps)
- Border animations on hover
- Lightbox with keyboard navigation
- Scanline overlay animation

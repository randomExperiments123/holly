# Design Document — Prototype 036

## Concept
Winter Ice captures the stillness and clarity of a frozen landscape. Every element feels crisp, cold, and clean — like breathing in winter air.

## Color Palette
- Ice blue: #e3f2fd
- Sky blue: #90caf9
- Deep freeze: #1565c0
- Pure white: #ffffff
- Text: #0d47a1 (dark), #546e7a (muted)

## Typography
- Headings: Playfair Display (serif, elegant)
- Body: Inter (sans-serif, clean)
- Hero heading: large, light weight, letter-spaced

## Layout
- Full-width sections with generous padding
- Two-column project grid on desktop, single on mobile
- Fixed frosted-glass navigation bar
- Cards with frosted glass effect (translucent backgrounds, backdrop blur)

## Key Visual Elements
- Snow-dot pattern via CSS radial-gradient background
- Subtle ice-blue gradients
- Frosted glass effect on project cards (backdrop-filter: blur)
- Rounded corners throughout
- Clean dividers between sections

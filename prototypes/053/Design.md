# Prototype 053 — Celtic: Design Decisions

## Color Palette
- `#1a3a1a` (dark green) — nav, headings
- `#2d5a2d` (forest green) — accents
- `#6b8e23` (olive) — secondary
- `#d4c5a9` (parchment) — background
- `#8b7355` (leather brown) — borders, text

## Typography
- Georgia / serif for body
- Decorative drop caps on headings
- Medieval-inspired letter spacing

## Layout
- Parchment-toned backgrounds
- CSS-generated knotwork border patterns (repeating gradients)
- Ornamental section dividers
- Project cards: dark green borders, gold accents

## Rationale
Celtic art's intricate knotwork and rich earth tones create a sense of craftsmanship and timelessness. The green/parchment palette evokes Irish manuscripts, while ornamental borders frame the portfolio content.

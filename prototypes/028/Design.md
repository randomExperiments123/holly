# Design Document — Prototype 028

## Concept
Clean, professional corporate design that prioritizes clarity and trust. The navy-and-blue palette conveys professionalism, while clean card layouts with subtle shadows create a polished portfolio presentation. This is design that means business.

## Color Palette
- Background: #f8fafc (ice white)
- Dark navy: #0f172a (for header, hero, footer)
- Primary blue: #2563eb (accents, buttons, links)
- Light blue: #dbeafe (subtle backgrounds)
- Text: #1e293b (dark slate)
- Muted text: #64748b (slate gray)
- Card bg: #ffffff

## Typography
- All text: 'Inter', sans-serif
- Professional, clean, highly readable
- Weight: 700 for headings, 500 for subheadings, 400 for body
- Tight letter-spacing for clean look
- Understated hierarchy

## Layout
- Clean white/light background
- Navy hero section with blue accents
- Centered max-width container (1100px)
- Two-column grid on desktop, single on mobile
- Solid nav bar with clean typography
- White cards with subtle box-shadows
- Small border-radius (8px) — professional but not harsh
- Blue accent borders and buttons
- Clean horizontal rules as dividers

## Key Interactions
- Card hover: subtle elevation with shadow increase
- Button hover: darker blue
- Clean, simple transitions (0.2s)
- Inline lightbox for project images
- Smooth scroll navigation

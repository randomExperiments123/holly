# PRD: Prototype 093 — Shadow Box

## Purpose
A dimensional, theatrical portfolio using layered CSS shadows, cut-out paper effects, and depth to create a shadow box / diorama aesthetic.

## Visual Identity
- **Concept**: Layered shadows, cut-out paper effect, depth through CSS shadows, dimensional, theatrical lighting
- **Primary palette**: Warm ivory (#faf3e0), deep charcoal (#2c2c2c), brass (#c5a55a), shadow blacks
- **Secondary palette**: Amber spotlight, parchment beige, dark walnut
- **Typography**: Elegant serif (Playfair Display), clean sans for body (Inter)
- **CSS Approach**: Multiple box-shadows for depth, text-shadows, perspective transforms, pseudo-element cutouts, spotlight gradients

## Content
Standard 5-project portfolio grid with image lightbox, hero section, about, contact. Same copy as baseline.

## Constraints
- Pure HTML + CSS + inline JS lightbox
- Mobile-first responsive
- AVIF images from `projects/projectN/`
- Accessible with ARIA, skip link, semantic HTML
- SEO meta tags + Schema.org VisualArtwork markup

# Design Document — Prototype 027

## Concept
Deconstructivism meets glitch art. The layout deliberately breaks traditional grid rules — sections skew at angles, elements overlap, and nothing sits perfectly straight. A CSS glitch effect on the main heading creates the signature visual. The dark, technical aesthetic evokes cyberpunk and digital decay.

## Color Palette
- Background: #0d0d0d (deep black)
- Secondary bg: #1a1a1a (dark grey)
- Cyan: #00f0ff (electric)
- Pink: #ff0066 (neon)
- Text: #e0e0e0
- Muted: #555555

## Typography
- All text: 'IBM Plex Mono', monospace (technical, code-like)
- Monospace throughout reinforces the digital/glitch theme
- Weight variations: 700 for headings, 400 for body
- Slight letter-spacing for readability

## Layout
- Broken/skewed grid — sections at slight angles
- Overlapping elements with negative margins
- Full dark background
- Centered container with overflow visible
- Two-column grid with intentional asymmetry
- Fixed header with glitch effect on logo
- Cards with diagonal clip-paths or transforms
- Raw, exposed edges (no rounding)

## Key Interactions
- CSS glitch animation on heading (pseudo-element offsets)
- Hover: scan-line effect
- Section hover: slight skew change
- Inline lightbox for project images
- Smooth scroll navigation

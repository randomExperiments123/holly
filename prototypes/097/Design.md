# Design Decisions — Prototype 097

## Concept
A vintage radio-inspired portfolio that feels like an antique tube radio cabinet. Warm amber glow, wood textures, brass accents, and speaker grille patterns create a nostalgic mid-century audio device aesthetic.

## Layout
- Centered hero with a large "radio dial" badge
- Project cards arranged in a speaker grille-like grid
- Circular frame cutouts for project images (like radio knobs)
- Warm radial glow emanating from behind content
- Vintage badge/emblem decorative elements

## Typography
"Playfair Display" for display headings evokes vintage print ads. Body in Georgia for classic serif readability. Monospace numerals for counter/dial displays.

## Color Palette
Deep wood browns (#3e2723, #1a0e0a) form the cabinet body. Warm amber (#ffb347) creates the tube glow. Cream (#fdf5e6) for text readability against dark backgrounds. Brass (#d4a574) for metallic accents.

## Decorative Elements
- CSS speaker grille pattern using radial-gradient circles in a grid
- Radio dial concentric circle decorations
- Warm glow radial gradients behind content
- Brass/gold line borders
- Vintage circular badge for the hero

## Lightbox
Dark wood-paneled backdrop, amber glow frame, brass-toned navigation buttons styled as radio dials.

## Accessibility Note
High contrast cream-on-dark text ensures readability. All animations respect prefers-reduced-motion.

- Amber glow: #ffb347
- Cream: #fdf5e6
- Brass: #d4a574

- Headings: "Playfair Display" (bold, vintage feel)
- Body: "EB Garamond" or Georgia (serif)
- Numbers: monospace (dial styling)

## Design Elements
- Speaker grille patterns (CSS grid of circles)
- Radio dial/knob decorative elements
- Warm radial gradient glow
- Brass/gold accent borders
- Vintage badge/circle frames
- Retro wave ornamental dividers

- 5 project cards with radio-aesthetic styling
- Working lightbox with keyboard navigation
- SEO meta tags + Schema.org
- Accessible (skip link, ARIA labels)
- Mobile-first responsive
- Grille pattern decoration via CSS

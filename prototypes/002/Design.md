# Design Document — Prototype 002

## Concept
Gravity-defying playfulness. Inspired by antigravity.google, this prototype breaks conventional layout rules. Elements float, overlap, and create a sense of joyful chaos.

## Color Palette
- Background: #0a0a2e (deep space blue)
- Accent 1: #ff6b6b (coral)
- Accent 2: #ffd93d (sun yellow)
- Accent 3: #6bcb77 (mint)
- Accent 4: #4d96ff (electric blue)
- Text: #ffffff, rgba(255,255,255,.7)

## Typography
- 'Space Grotesk' from Google Fonts
- Variable weights for kinetic effects
- Large, bold headings with letter-spacing play
- Mixed case and rotated text elements

## Layout
- Asymmetrical, overlapping sections
- Floating card elements with hover levitation
- Diagonal section dividers
- Hero with scattered, orbiting text elements
- Gradient blobs as background decoration

## Key Interactions
- Cards rise on hover (translateY with shadow)
- Gradient background animation
- Smooth scroll navigation
- Lightbox for project images

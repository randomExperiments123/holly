# Design Document — Prototype 043

## Concept
Steampunk reimagines the portfolio as a Victorian engineer's folio. Brass fittings, leather-bound books, gear mechanisms, and steam-powered aesthetics create an antique industrial feel.

## Color Palette
- Dark wood: #2d1b0e (background)
- Bronze: #4a3728
- Brass: #8b7355
- Teal steam: #00bcd4
- Parchment: #f5e6c8 (text areas)
- Gold accent: #c9a84c
- Text: #e8dcc8 (warm light)

## Typography
- Headings: Cinzel (decorative serif, Victorian)
- Body: Raleway (clean, industrial sans)
- Ornamented ampersands and numerals

## Layout
- Ornate framed sections
- Parchment-textured content areas
- Gear-corner decorations
- Symmetrical, formal layout
- Brass-rule dividers

## Key Visual Elements
- Gear/cog pattern via CSS radial-gradients and pseudo-elements
- Ornate border frames (double-line, corner ornaments)
- Parchment texture backgrounds
- Brass/gold metallic gradients
- Victorian decorative rules (:<hr> with ornaments)
- Compass-rose or gear motifs in corners

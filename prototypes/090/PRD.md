# PRD: Stripes Only — 090

## Purpose
A maximalist, pattern-on-pattern exploration where every visual element is constructed from stripes — horizontal, vertical, diagonal. Bold graphic design that celebrates the stripe as the sole decorative atom.

## Requirements
- Every section background uses a stripe pattern (repeating-linear-gradient)
- Multiple stripe directions: horizontal, vertical, 45°, -45°
- Pattern-on-pattern layering (cards on striped backgrounds with different stripe orientations)
- High contrast palette: black, white, one accent (orange-red)
- Bold, loud, graphic feel
- Responsive and mobile-first
- Inline lightbox with keyboard nav

## Constraints
- Pure HTML + CSS + inline JS
- No gradients, solids, or images except project AVIFs
- Stripes are the only decorative element permitted

## Success Criteria
- All backgrounds, buttons, dividers use stripe patterns
- Pattern mixing is visually coherent (not chaotic)
- Text remains readable on busy stripe backgrounds
- Mobile layout preserves the stripe-first identity
- Feels intentional and editorial, not noisy

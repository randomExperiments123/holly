# Design Document — Prototype 023

## Concept
Bringing the tranquility of nature to digital portfolio design. Deep forest greens anchor the layout while sage and cream tones soften and warm. Organic shapes, natural color gradients, and leafy decorative elements create an earthy, sustainable feel.

## Color Palette
- Primary: #2d5016 (deep forest green)
- Secondary: #a8c686 (sage green)
- Background: #f5f0e8 (warm cream)
- Accent: #4a7c2e (mid green)
- Text: #1a2e0d (dark green-black)
- Light text: #6b8f4e

## Typography
- Headings: 'Cormorant Garamond', serif (elegant, natural)
- Body: 'Inter', sans-serif (clean, readable)
- Cormorant Garamond brings organic, literary feel
- Inter for clean body text contrast

## Layout
- Cream background with green accents
- Centered max-width container (1100px)
- Two-column grid on desktop, single on mobile
- Forest green fixed header with cream text
- Cards with cream backgrounds, green borders
- Decorative leaf-inspired separators
- Organic rounded corners (20px)
- Natural shadow effects

## Key Interactions
- Card hover: gentle green glow
- Subtle scale on hover
- Inline lightbox for project images
- Smooth scroll navigation

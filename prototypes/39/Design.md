# Design Document — Prototype 039

## Concept
Tech Noir channels the dark, neon-lit atmosphere of 1980s cyberpunk cinema. The portfolio feels like a terminal interface — cold, technological, and tinged with danger.

## Color Palette
- Deep teal: #0a1628 (background)
- Hot pink: #ff2a6d (primary accent)
- Electric blue: #05d9e8 (secondary accent)
- Neon white: #f0f0f0 (text)
- Dim text: #4a6a8a (muted)

## Typography
- Share Tech Mono (monospace, all text)
- Terminal-style uppercase headings
- Code-like project numbering

## Layout
- Dark full-bleed sections
- Grid-based project layout
- Neon borders on cards
- Scanline overlay on backgrounds
- CRT-style glow effects

## Key Visual Elements
- Scanline CSS animation (repeating gradient)
- Neon glow via box-shadow with hot pink/electric blue
- Grid-line background resembling terminal grids
- Blinking cursor accent elements
- VHS static noise effect on hero
- Data-stream line animation

# Design Document — Prototype 004

## Concept
Feminine elegance. Soft, warm, and inviting — this prototype uses gentle colors, graceful serif typography, and delicate decorative elements to create an intimate portfolio experience.

## Color Palette
- Background: #faf0f5 (soft rose white)
- Primary text: #2d1b2e (deep plum)
- Secondary text: #8b6f8e (muted mauve)
- Accents: #e8a0bf (rose), #d4a0b0 (dusty rose), #c4b5d4 (lavender)
- Cards: #ffffff with subtle shadow

## Typography
- Headings: 'Playfair Display' (Google Fonts) — elegant serif
- Body: 'Inter' — clean sans-serif contrast
- Italic for emphasis and decorative quotes
- Refined letter-spacing for headings

## Layout
- Full-width header with decorative flourishes
- White cards with rounded corners and soft shadows
- Vertical rhythm with generous spacing
- Centered text with asymmetrical visual elements
- Signature-style logo mark
- Curved decorative SVG dividers between sections

## Key Interactions
- Gentle fade-in on scroll
- Card hover with subtle lift and glow
- Lightbox for project images

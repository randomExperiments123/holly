# Design Document — Prototype 035

## Concept
Tropical — the portfolio embodies a tropical paradise. Bright sun, ocean waves, palm-like decorative elements, and a vibrant color palette inspired by beach destinations.

## Color Palette
- Background: #fff8e7 (warm sunlight)
- Primary: #ffd700 (sun yellow)
- Ocean: #00b894 (teal)
- Accent: #ff6b6b (hibiscus pink)
- Dark: #2d3436 (palm green/leaf)
- Section: full-bleed color blocks

## Typography
- 'Inter' (Google Fonts) — clean, highly readable sans-serif
- Bold weight for headings with generous tracking
- Regular weight for body text

## Layout
- Full-bleed hero with tropical gradient
- Alternating section background colors
- Cards with rounded corners and subtle shadows
- Decorative wave dividers between sections via CSS
- Large, generous spacing

## Key Interactions
- Sunny warm glow animations
- Wave-like motion on decorative elements
- Vibrant hover color shifts
- Lightbox with keyboard navigation

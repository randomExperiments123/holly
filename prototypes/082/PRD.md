# PRD: 082 — Loud Typography

## Overview
A typography-first portfolio where text is the hero. Massive scale, kinetic word cascades, bold color contrasts, and minimal imagery. Type does all the talking.

## Requirements
- Hero section: single massive word or phrase filling the viewport
- Kinetic/animated text — words that cascade, slide, or stagger on load
- Bold color contrasts (neon on dark or extreme light/dark)
- Minimal project imagery — words take center stage
- Type scale ranges from 14px to 20vw
- Standard lightbox for project images
- Mobile-first, responsive, accessible

## Content Structure
1. Hero — Massive "HOLLY" or "DESIGN" fills screen, sub text smaller
2. Intro — "Words create worlds" thematic intro in varying sizes
3. Projects — cards with large typographic treatment + small image
4. About — typographic portrait (type-based self-description)
5. Contact — bold CTA
6. Footer

## Constraints
- Pure HTML+CSS+inline JS
- AVIF images only
- No external dependencies
- Must pass accessibility checks

# Design Document — Prototype 031

## Concept
Watercolor — the portfolio simulates a watercolor sketchbook. Each section appears painted on paper with soft bleeding edges and transparent color washes.

## Color Palette
- Paper: #faf5f0 (warm cream)
- Washes: #b8d4e3 (dusty blue), #d4a5b5 (rose), #c9d8b6 (sage green)
- Text: #5a4a3a (warm brown)
- Accent: #c4956a (warm gold)

## Typography
- Playfair Display italic (Google Fonts) — elegant serif with beautiful swashes
- System sans-serif for UI elements
- Italic headings for a hand-painted feel

## Layout
- Cards with "painted edge" effect using CSS blur + border-radius
- Soft drop shadows that mimic water pooling
- Asymmetric grid for an organic feel
- Warm paper-toned backgrounds

## Key Interactions
- Subtle float animation on cards (like paper drifting)
- Watercolor drip-border hover effects
- Lightbox with keyboard navigation
- Smooth scroll

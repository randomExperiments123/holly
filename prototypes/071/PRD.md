# PRD: Terminal / Hacker Aesthetic Portfolio

## Purpose
Present Holly Matthews' graphic design portfolio in a retro terminal/hacker theme. Green-on-black CRT monitor aesthetic targeting tech-savvy recruiters and studios.

## Requirements
- Monospace typography throughout (Courier New / SF Mono)
- Green (#00ff00) text on black (#000) background
- CRT scanline overlay via CSS pseudo-element
- Blinking block cursor animation
- Command-line prompts ($) before headings
- ASCII art in hero section
- Keyboard-interactive feel (typewriter-style reveals optional)
- Lightbox maintains terminal look: green borders, monospace counter

## Content
Same 5 projects as all prototypes with project images in AVIF format.

## Constraints
- Pure HTML+CSS, inline JS only
- Mobile-first responsive
- Accessible (ARIA, skip link, semantic HTML)
- AVIF images only
- Schema.org markup for VisualArtwork
- No external dependencies

# Design Document — Prototype 010

## Concept
Magazine editorial — a sophisticated, typography-first design that treats each project like a magazine feature article.

## Color Palette
- Background: #f8f6f3 (warm white)
- Text: #1a1a1a (body), #999 (secondary)
- Accent: #c9a84c (gold), #1a1a1a
- Pull quote: #c9a84c with large serif

## Typography
- Headings: 'Playfair Display' — elegant serif
- Body: 'Inter' — clean sans-serif
- Feature article styling with drop caps
- Italic pull quotes
- 11px uppercase captions

## Layout
- Magazine-style spreads
- Two-column project descriptions
- Full-bleed hero image area
- Pull quotes highlighted with gold accent
- Article-style about section
- Table of contents-style navigation

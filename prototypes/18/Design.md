# Design Document — Prototype 018

## Concept
Modern tech/SaaS aesthetic. Clean, professional, and trustworthy — like a Stripe or Linear landing page. The portfolio positions the designer as a polished, modern creative professional.

## Color Palette
- Background: #ffffff (pure white)
- Text: #0f172a (dark navy)
- Accent: #2563eb (vibrant blue)
- Secondary text: #64748b (slate)
- Borders: #e2e8f0 (light slate)

## Typography
- Font: 'Inter' (Google Fonts), all weights
- 600 weight for navigation and buttons
- Large, bold hero heading (64px)
- Clean, technical feel — no serifs

## Layout
- Full-width hero with gradient background
- Three-column project grid on desktop, two on tablet, one on mobile
- Fixed white header with subtle bottom border
- Card hover: lift + shadow
- Border-radius: 12px on cards, 8px on buttons
- Generous use of box-shadow for depth

## Key Interactions
- Card hover lift with enhanced shadow
- Button hover with slight darkening
- Inline lightbox for project images
- Smooth scroll navigation

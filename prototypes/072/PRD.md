# PRD: Victorian Aesthetic Portfolio

## Purpose
Present Holly Matthews' portfolio with an ornate Victorian/antique design. Burgundy, cream, and gold palette with decorative serif typography and filigree ornamentation.

## Requirements
- Serif typography (Playfair Display, Georgia)
- Burgundy (#800020), cream (#FFFDD0), gold (#D4AF37) palette
- Ornate decorative borders via CSS
- Filigree ornamental elements
- Vintage wallpaper background patterns (CSS-only)
- Antique frame effects around project images
- Victorian-era decorative dividers between sections
- Lightbox with gilt-frame aesthetic

## Content
Same 5 projects with AVIF images.

## Constraints
- Pure HTML+CSS, inline JS only
- Mobile-first responsive
- Accessible (ARIA, skip link, semantic HTML)
- AVIF images only
- Schema.org markup
- No external dependencies

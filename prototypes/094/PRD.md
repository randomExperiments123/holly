# PRD: Prototype 094 — Neon Sign

## Purpose
A glowing, electrified portfolio inspired by neon tube signage, glass-like effects, and buzzing urban night energy.

## Visual Identity
- **Concept**: Glowing neon tube typography, dark backgrounds, glass-like effects, buzzing energy
- **Primary palette**: Hot pink (#ff2d95), electric blue (#00d4ff), neon green (#39ff14), on black (#0a0a0a)
- **Secondary palette**: Amber neon, ultraviolet purple
- **Typography**: Display script/rounded (Pacifico or similar), monoline sans for body
- **CSS Approach**: Multiple text-shadows for neon glow, blur filters for bloom, flicker animations, glass/reflection pseudo-elements, dark background

## Content
Standard 5-project portfolio grid with image lightbox, hero section, about, contact. Same copy as baseline.

## Constraints
- Pure HTML + CSS + inline JS lightbox
- Mobile-first responsive
- AVIF images from `projects/projectN/`
- Accessible with ARIA, skip link, semantic HTML
- SEO meta tags + Schema.org VisualArtwork markup

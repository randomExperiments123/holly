# PRD — Prototype 006: Swiss International

## Overview
Swiss/International Style portfolio. Clean grid, sans-serif typography, red accent, photography-focused layout.

## Design Direction
- Swiss/International typographic style
- Strict grid system
- Helvetica or similar grotesk typeface
- Red (#ff0000) as accent color
- Black and white with red highlights
- Asymmetric grid layouts
- Objective, clean, functional

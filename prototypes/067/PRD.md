# PRD: Soviet Constructivism (067)

## Purpose
Present Holly Matthews' portfolio in the bold, propagandistic visual language of early Soviet Constructivism. Inspired by El Lissitzky, Rodchenko, and Stalinist-era posters — red/black/white palette, angular geometry, photomontage aesthetic.

## Requirements
- Bold red (#cc0000), black (#1a1a1a), white (#f0f0f0) colour palette
- Diagonal split backgrounds, angular typography
- Hammer & sickle decorative CSS motifs (non-political, purely aesthetic)
- Photomontage-style overlapping elements
- Heavy, blocky typography with slanted angles
- Maintain all 5 project cards, lightbox, skip link, SEO, Schema.org, ARIA
- Mobile-first responsive layout

## Constraints
- Pure HTML + CSS, zero JS dependencies (except inline lightbox)
- AVIF images only
- Self-contained
- No external fonts

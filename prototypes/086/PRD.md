# PRD: Variable Font Play — 086

## Purpose
Explore typography as performance using variable font axes. Every interaction triggers a shift in weight, width, or slant, making the type itself the primary visual experience.

## Requirements
- Load at least one variable font with multiple axes (wght, wdth, slnt, opsz)
- Animate font-weight on hover for navigation, project cards, and section headings
- Hero title animated continuously through weight spectrum (200–900)
- Project cards show weight/width shift on hover
- Responsive and mobile-first
- Accessible — reduced motion media query pauses all animations
- Schema.org markup on projects
- Inline lightbox with keyboard nav

## Constraints
- Pure HTML + CSS + inline JS (lightbox only)
- AVIF images from shared assets
- Zero external dependencies except Google Fonts variable font

## Success Criteria
- Typography feels alive and responsive
- Each section has a distinct typographic treatment
- Smooth performance on mobile devices

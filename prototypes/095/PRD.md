# PRD: Prototype 095 — Cartoon

## Purpose
A fun, bold portfolio inspired by comic books — thick outlines, Ben-Day dots, speech bubbles, and primary-color pop.

## Visual Identity
- **Concept**: Comic book style, thick outlines, Ben-Day dots, speech bubbles, primary colors, fun bold
- **Primary palette**: Comic red (#e23636), primary blue (#00aaff), yellow (#ffd700), black outlines (#1a1a1a)
- **Secondary palette**: Fighting green, purple, white
- **Typography**: Bold comic style (Bangers), comic book sound effects vibe
- **CSS Approach**: Thick borders (3-4px black), Ben-Day dot repeating radial gradients, speech bubble pseudo-elements, halftone patterns, outlined text effects, bold shadows

## Content
Standard 5-project portfolio grid with image lightbox, hero section, about, contact. Same copy as baseline.

## Constraints
- Pure HTML + CSS + inline JS lightbox
- Mobile-first responsive
- AVIF images from `projects/projectN/`
- Accessible with ARIA, skip link, semantic HTML
- SEO meta tags + Schema.org VisualArtwork markup

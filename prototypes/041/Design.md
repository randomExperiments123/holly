# Design Document — Prototype 041

## Concept
Zen Calm embodies the serenity of a Zen garden. Every element is intentional, minimal, and quiet. The design breathes — spacious, centered, and deeply peaceful.

## Color Palette
- Warm stone: #f0ebe3 (background)
- Sage: #8a9b8e (primary accent)
- Deep moss: #5a6b5e (dark accent)
- Sand: #e5ddd0 (darker stone)
- Text: #3d4a3d (dark moss)
- Lighter sage: #b5c4b8

## Typography
- Headings: Noto Serif JP (serif, elegant Japanese)
- Body/UI: Inter (clean, understated)
- Small, refined sizes
- Generous line-height

## Layout
- Single-column throughout (even on desktop)
- Narrow content width (max 680px)
- Vertical rhythm with generous spacing
- Everything centered
- No fixed navigation — simple top links

## Key Visual Elements
- Zen garden texture via CSS gradient
- Minimal line separators (single pixel)
- Rounded stone-like buttons
- Subtle hover transitions
- No shadows, no heavy borders
- Natural, organic feel

# Design Document — Prototype 014

## Concept
Art Deco luxury. Geometric patterns, gold ornamentation, and symmetrical layouts evoke the glamour of 1920s design. The portfolio feels like a high-end jewellery catalogue or a vintage film title sequence.

## Color Palette
- Background: #0d0d1a (dark navy/black)
- Surface: #1a1a2e (deep navy)
- Accent: #d4af37 (gold)
- Text: #f0e6d3 (warm white)
- Secondary: #c0a060 (darker gold)

## Typography
- Headings: 'Cinzel' serif (Google Fonts) — decorative, all-caps
- Body: 'Raleway' sans-serif (Google Fonts)
- Hero: clamp(40px, 7vw, 64px), Cinzel, all caps
- Body: 15px, light weight

## Layout
- Strong symmetry throughout
- Geometric border decorations via CSS
- Fan/chevron patterns as section dividers
- Project cards with gold border frames
- Centered, column-based layout
- Luxurious spacing (generous padding)

## Key Interactions
- Gold shimmer on hover
- Corner ornament decoration on cards
- Fade transitions
- Standard lightbox for project images

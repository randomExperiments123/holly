# Design Document — Prototype 005

## Concept
Dark neubrutalism — bold, unapologetic, high-contrast. Inspired by brutalist web design with modern dark mode sensibilities.

## Color Palette
- Background: #0d0d0d
- Surface: #1a1a1a
- Text: #ffffff, #a0a0a0
- Accent: #ff3b30 (red), #ffd60a (yellow)
- Borders: 3px solid #fff

## Typography
- 'DM Sans' (Google Fonts) — bold, geometric
- 900 weight for headings, 700 for subheadings
- 4px solid underlines
- Large scale: 96px hero, 24px body

## Layout
- Asymmetric grid
- Full-bleed sections
- Staggered project cards
- Thick borders everywhere
- Raw, unpolished alignment

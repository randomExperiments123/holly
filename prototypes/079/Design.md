# Design Document — Prototype 079: Christmas

## Design Direction
Warm, festive, cozy. Inspired by traditional Christmas colors, crackling fires, and holiday cheer.

## Color Palette
- **Crimson Red**: #dc143c — primary accent
- **Emerald Green**: #2e8b57 — headings, dark accents
- **Gold**: #d4af37 — decorative elements, highlights
- **Warm Cream**: #fdf6e3 — background
- **Forest**: #1a3c2a — text, footer
- **Holly Red**: #cc0000 — button accents

## Typography
- **Headings**: Festive serif `'Palatino', 'Georgia', serif` with gold styling
- **Body**: `'Merriweather', 'Georgia', serif` — warm readable serif
- **Accents**: Small caps with letter-spacing

## Key Visual Elements
1. **Snowflake patterns**: CSS repeating patterns using radial-gradient and conic-gradient
2. **Holly motifs**: Green pseudo-element leaves with red berry dots
3. **Warm glow**: Golden box-shadows and background gradients
4. **Ribbon-like accents**: Decorative horizontal bars with gold borders
5. **Cozy texture**: Subtle warm gradient overlays

## Layout
- Single-column mobile, two-column desktop
- Rounded corners with gold double-borders
- Centered layout with warm, inviting proportions

## Lightbox
Gold and red navigation buttons. Warm backdrop blur. Festive counter styling.

## Accessibility
- Skip link
- ARIA labels
- High contrast for text readability
- Gold focus indicators

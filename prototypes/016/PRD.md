# Product Requirements Document — Prototype 016

Scandinavian minimal portfolio for Holly Matthews. Warm, cozy, hygge-inspired design with clean lines and generous whitespace.

- Inspiration: Scandinavian interior design, IKEA catalogues, hygge aesthetic

- Cozy, inviting feel

- Clean lines, minimal ornamentation

- 5 design projects: Nordica Coffee, Urban Perspectives, Motion & Interaction, Botanic, Dreamscapes
- About section
- Contact section
- Download portfolio button

## Technical Requirements
- Pure HTML + CSS, zero JS dependencies
- Inline lightbox with minimal inline JS
- AVIF images
- Responsive mobile-first
- Accessible
- SEO-optimized with Schema.org

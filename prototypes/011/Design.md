# Design Document — Prototype 011

## Concept
Cyberpunk aesthetic. The portfolio channels retro-futuristic digital culture with neon glow, scan lines, and data-stream visuals. Every element feels like it belongs in a terminal or HUD interface.

## Color Palette
- Background: #0a0a0a (deep black)
- Primary accent: #00ff41 (neon green)
- Secondary accent: #ff00ff (neon magenta)
- Text: #e0e0e0 (light gray), #00ff41 (green highlights)
- Borders: rgba(0, 255, 65, 0.3)

## Typography
- 'Share Tech Mono' from Google Fonts (all text)
- Monospace throughout for terminal feel
- Hero: clamp(36px, 8vw, 72px)
- Body: 15px

## Layout
- Full-height sections
- Grid background pattern (horizontal + vertical lines)
- Fixed navigation with neon underline
- Single column project cards with neon borders
- Hover: glow effect on cards

## Key Interactions
- Glitch animation on hero heading (CSS keyframes)
- Scan-line overlay on hover
- Neon glow pulse on interactive elements
- Standard lightbox for project images

# Prototype 051 — Vintage 70s

## Purpose
Demonstrate a graphic design portfolio with a retro 1970s aesthetic. Warm, nostalgic, earthy.

## Target Vibe
Groovy, warm, nostalgic, earthy, organic. Think harvest gold, avocado green, burnt orange.

## Sections
- Hero with retro typography and patterned background
- Intro with era-appropriate voice
- 5 project cards with rounded badges and warm tones
- About section
- Contact section
- Footer
- Lightbox for image viewing

## Design Constraints
- Pure HTML + CSS, inline lightbox JS
- AVIF images only
- Mobile-first responsive
- Accessible (ARIA, skip link, semantic HTML)
- SEO meta tags + Schema.org

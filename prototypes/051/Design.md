# Prototype 051 — Vintage 70s: Design Decisions

## Color Palette
- `#5c3a21` (dark brown) — nav background, text
- `#d4813d` (burnt orange) — hero gradient, accents
- `#e8a838` (golden yellow) — buttons, highlights
- `#f5e6c8` (cream) — background
- `#2d5016` (avocado green) — secondary accents

## Typography
- Headings: serif (Georgia) for warm, retro feel
- Body: system sans-serif for readability
- Decorative ampersands and swashes

## Layout
- Centered, max-width constrained
- Rounded corners throughout
- Groovy wavy/geometric decorative borders via CSS
- Badge-style project numbering (circle badges)

## Rationale
The 1970s aesthetic is immediately recognizable through its earthy color palette and rounded, friendly shapes. This prototype leans into nostalgia with warm gradients, badge elements, and a cream backdrop that evokes aged paper.

# Design Decisions — Prototype 044

## Concept
A sleek, futuristic portfolio that evokes the feeling of an AI agent interface. Dark backgrounds with purple and cyan neon accents create a tech-forward, slightly mysterious atmosphere.

## Layout
- Single column hero with animated gradient text
- Two-column project grid (single column on mobile)
- Glass cards with subtle backdrop blur
- Fixed navigation bar with blur effect

## Typography
Inter was chosen for its clean, technical feel that complements the AI/cyber theme.

## Lightbox
Standard overlay lightbox adapted to the dark theme with glass-morphism controls and purple/cyan accent glows.

## Key Visual Effects
- Animated gradient background on hero
- Glow borders on card hover
- Scanning line animation across hero section
- Grid pattern overlay reminiscent of tech interfaces

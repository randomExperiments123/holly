# Prototype 058 — Claymorphism

## Overview
A portfolio page using the claymorphism design trend — soft 3D extruded shapes, warm earthy colours, extreme rounding, and plush shadow depth.

## Requirements
- All 5 projects with image slideshow
- Lightbox for full-screen image viewing
- Mobile-first responsive layout
- Accessible (ARIA, skip link, semantic HTML)
- SEO meta tags + Schema.org markup
- Pure HTML + CSS, inline lightbox JS only

## Design Direction
- Soft 3D extruded shapes with dual-layer shadows
- Warm earthy colour palette (terracotta, ochre, clay, cream)
- Extreme border-radius on all elements
- Pillowy, tactile visual feel
- Rich depth through multiple box-shadow layers

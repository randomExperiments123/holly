# Prototype 052 — De Stijl

## Purpose
Portfolio inspired by the De Stijl movement (Mondrian, Van Doesburg). Strict geometric abstraction.

## Target Vibe
Bold, minimalist, primary-colored, grid-based, artistic. Black lines, red/blue/yellow blocks.

## Sections
- Hero with Mondrian-style grid layout
- Intro
- 5 project cards with thick black borders, color blocks
- About
- Contact
- Footer
- Lightbox

## Design Constraints
- Pure HTML + CSS, inline lightbox JS
- AVIF images only
- Mobile-first responsive
- Accessible
- SEO + Schema.org

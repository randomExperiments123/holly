# Prototype 052 — De Stijl: Design Decisions

## Color Palette
- `#f5f5f5` (off-white) — main background
- `#1a1a1a` (black) — borders, text, grid lines
- `#e30613` (red) — primary accent
- `#0039a6` (blue) — secondary accent
- `#ffd700` (yellow) — tertiary accent

## Typography
- Helvetica/Arial, bold weights
- All-caps section labels
- Clean, geometric, sans-serif only

## Layout
- Thick black borders on everything (8px+)
- Rectangular color block accents
- Asymmetric hero layout with Mondrian-style grid cells
- Project cards: bordered, color-block tags

## Rationale
De Stijl's radical simplicity translates perfectly to a portfolio: content is organized into clear, bold rectangles. The primary color accents draw immediate attention to project cards and CTAs, while black grid lines create strong visual structure.

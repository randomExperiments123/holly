# Design Document — Prototype 013

## Concept
Organic nature-inspired design. Soft earth tones, flowing organic shapes, and botanical decorative elements create a warm, grounded portfolio experience. Serif headings evoke natural forms while clean sans-serif body ensures readability.

## Color Palette
- Primary: #2d5a27 (deep forest green)
- Secondary: #8cb369 (sage green)
- Background: #f4e4c1 (warm cream)
- Text: #3a2e1f (dark brown)
- Accent: #c4a77d (tan/gold)

## Typography
- Headings: 'Cormorant Garamond' serif (Google Fonts)
- Body: 'Inter' sans-serif (Google Fonts)
- Hero: clamp(44px, 8vw, 80px), serif, italic
- Body: 16px, light weight

## Layout
- Rounded corners throughout
- Organic flowing section transitions via border-radius
- Project cards with soft shadows and leaf corner decorations
- Asymmetric but balanced layout
- Botanical decorative elements via CSS ::before/::after

## Key Interactions
- Soft hover lift with leaf-shadow effect
- Gentle transitions (0.4s ease)
- Natural scroll behavior
- Standard lightbox for project images

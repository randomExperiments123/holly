# Design Document — Prototype 034

## Concept
Sci-Fi Futuristic — the portfolio presents work through a holographic HUD interface. Cards appear as floating glass panels with glowing borders and subtle tech patterns.

## Color Palette
- Background: #0b0b1a (deep void blue-black)
- Primary: #4fc3f7 (electric blue)
- Secondary: #7c4dff (quantum purple)
- Accent: #00e676 (neon green for data elements)
- Glass: rgba(255,255,255,0.05) with backdrop blur

## Typography
- 'Orbitron' (Google Fonts) — futuristic geometric sans-serif
- Variable font weights for hierarchy
- Generous letter-spacing for headings

## Layout
- Cards with glass-effect backdrop blur and subtle white borders
- Grid lines and data readout decorative elements
- Circular progress indicators as design motifs
- Fixed translucent nav with glass effect

## Key Interactions
- Holographic shimmer animation on card hover
- Pulsing glow on interactive elements
- Border gradient animations
- Lightbox with keyboard navigation

# PRD: Pixel Art 2 (068)

## Purpose
Display Holly Matthews' portfolio through an 8-bit retro gaming aesthetic. Inspired by NES-era graphics with extreme pixelation, grid overlays, scanline effects, and a bright NES colour palette.

## Requirements
- NES-inspired colour palette: #0f0f0f (black), #e4000f (red), #f8a000 (orange), #f8d800 (yellow), #00a800 (green), #0040c0 (blue), #f0f0f0 (white)
- Scanline overlay effect across entire page
- Pixel grid background pattern
- Blocky, monospace typography
- Pixelated borders (simulated with box-shadow steps)
- Maintain all 5 project cards, lightbox, skip link, SEO, Schema.org, ARIA
- Mobile-first responsive layout

## Constraints
- Pure HTML + CSS, zero JS dependencies (except inline lightbox)
- AVIF images only
- Self-contained
- No external fonts

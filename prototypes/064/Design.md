# Design Document — Prototype 064

## Concept
Biomechanical fusion. Organic flesh meets industrial machinery. The portfolio feels alive — pulsing, breathing, wired. A dark sci-fi interface that's part organism, part machine.

## Color Palette
- Background: #0D0806 (deep organic black), #1A0F0A (warm dark)
- Text: #D4C5B0 (bone), #8B7D71 (aged metal)
- Accents: #D4A574 (flesh), #8B5E3C (deep tissue), #A8A8A8 (steel), #4A4A4A (iron)
- Highlights: #C0392B (blood vessel red), #5DADE2 (coolant blue)

## Typography
- "IBM Plex Sans" (Google Fonts) — technical, clean, industrial
- "Syne Mono" (Google Fonts) — mechanical monospace for counters
- "Abril Fatface" for section titles — organic curves contrast
- Technical labels styled as circuit traces

## Layout
- Hero with organic-mechanical split composition
- Project cards with "grafted" borders (organic + mechanical)
- Vein/wire SVG patterns framing sections
- Section backgrounds alternate between "tissue" (warm organic) and "metal" (cool industrial)
- Radiating concentric circles resembling bioreactor rings
- Footer as "maintenance panel"

## Key Interactions
- Pulse animation on organic elements
- Mechanical click response on interactive elements
- Lightbox with tube/frame border
- Wire trace animation on hover
- Scanning line effect on images

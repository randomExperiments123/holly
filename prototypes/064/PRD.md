# Product Requirements Document — Prototype 064

## Concept
Biomechanical/organic-mechanical hybrid portfolio. Flesh tones, metal surfaces, visible veins and wires. Dark sci-fi aesthetic blending organic curves with industrial machinery.

## Design Direction
- Inspiration: H.R. Giger, biomechanical art, organic machinery
- Flesh tones (#D4A574, #8B5E3C) + metal (#A8A8A8, #4A4A4A)
- Vein/wire patterns as decorative elements
- Dark moody background (#0D0806)
- Organic curves intersecting with rigid mechanical lines
- Tissue-like textures via CSS gradients

## Content
- 5 design projects with the standard project data
- About section
- Contact section
- Download portfolio button

## Technical Requirements
- Pure HTML + CSS, zero JS dependencies
- Inline lightbox with minimal inline JS
- AVIF images
- Responsive mobile-first
- Accessible
- SEO-optimized with Schema.org

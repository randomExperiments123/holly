# Design Decisions — Prototype 046

## Concept
Inspired by Gothic cathedral architecture — pointed arches, rose windows, and ornate stonework translated into web design. The result is dramatic, dark, and deeply atmospheric.

## Layout
- Full-screen hero with pointed arch framing
- Single-column project list
- Decorative arch borders on cards
- Stained glass gradient headers

## Typography
Cinzel Decorative is used for major headings to evoke carved stone lettering. Cormorant Garamond provides readable body text with elegant serif proportions.

## Key Visual Effects
- Pointed arch shapes created with CSS border-radius and clip-path
- Stained glass effect via multi-stop radial gradients
- Gold metallic text effect for headings
- Ornamental dividers between sections
- Subtle parchment texture on light elements

## Lightbox
Dark stained-glass style frame with gold-toned controls and ornate border details.

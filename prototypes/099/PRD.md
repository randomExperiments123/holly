# Product Requirements Document — Prototype 099

## Theme
Futurist — sleek sci-fi aesthetic, year 3000 vibe, metallic gradients, neon accents

## Color Palette
- Deep space: #0a0a1a
- Metallic silver: #c0c0c0
- Neon cyan: #00f0ff
- Neon magenta: #ff00ff
- Glass white: rgba(255,255,255,0.08)
- Dark purple: #1a0a2e

## Typography
- Headings: "Orbitron" (geometric sans, futuristic)
- Body: "Exo 2" (sans-serif, tech feel)
- Monospace for data-like accents

## Design Elements
- Glassmorphism cards (frosted glass effect)
- Neon glow borders and shadows
- Floating/levitating card animations
- Grid line background pattern
- Metallic gradient text
- Glow pulse animations on accents

## Requirements
- 5 project cards with sci-fi styling
- Working lightbox with keyboard navigation
- SEO meta tags + Schema.org
- Accessible (skip link, ARIA labels)
- Mobile-first responsive
- Neon glow and glass effects via CSS

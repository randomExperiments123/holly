# Design Decisions — Prototype 099

## Concept
A futuristic sci-fi portfolio set in the year 3000. Sleek metallic surfaces, neon cyan and magenta accents, floating glass cards, and grid-line cyberpunk-lite aesthetics create a tech-forward, otherworldly feel.

## Layout
- Full dark space with perspective grid background
- Floating/levitating glassmorphism project cards
- Neon-bordered sections with glow effects
- Holographic-style hero with glitch-lite text treatment
- Asymmetrical, dynamic layout grid

## Typography
"Orbitron" in various weights for an unmistakably futuristic heading look. "Exo 2" for clean tech body text. Monospace for data readout accents.

## Color Palette
Deep space (#0a0a1a) and dark purple (#1a0a2e) form the cosmic void. Metallic silver (#c0c0c0) for text. Neon cyan (#00f0ff) and magenta (#ff00ff) for electric accents. Glass white for transparency layers.

## Decorative Elements
- Glassmorphism (backdrop-filter blur, semi-transparent backgrounds)
- Neon glow via multiple text-shadows and box-shadows
- Grid line background using repeating-linear-gradient
- Floating animation (levitate keyframes)
- Metallic gradient text (silver-to-white)
- Scan-line overlay effect

## Lightbox
Neon-glowing border, dark glass backdrop, cyan-accented controls with glow hover effects.

## Accessibility Note
High contrast metallic text against dark backgrounds. All animations respect prefers-reduced-motion.

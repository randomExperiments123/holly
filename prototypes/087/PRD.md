# PRD: Duotone — 087

## Purpose
Create a high-contrast, poster-like portfolio using a strict two-color palette with halftone dot textures. Evokes screen-printed rock posters and avant-garde editorial.

## Requirements
- Strict two-color palette (hot pink + blue) applied to all elements
- Halftone dot patterns via CSS radial-gradient backgrounds
- Images rendered in duotone via CSS mix-blend-mode and filter chains
- Large, bold, poster-scale typography
- High contrast ratios for accessibility
- Responsive and mobile-first
- Inline lightbox with keyboard nav

## Constraints
- Pure HTML + CSS + inline JS
- No images or SVGs beyond the project AVIFs
- Halftone must be pure CSS (no canvas or SVG)

## Success Criteria
- Every element conforms to the two-color rule
- Halftone texture is visible but not distracting
- Images read as duotone (shadows one color, highlights the other)
- Poster feel translates to mobile

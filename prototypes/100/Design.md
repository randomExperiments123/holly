# Design Decisions — Prototype 100

## Concept
A solarpunk portfolio celebrating an optimistic, green-tech future. Sunshine yellows, sky blues, and leaf greens combine with organic flowing shapes and clean modern design. The hundredth prototype radiates hope and community.

## Layout
- Bright, airy hero with sunburst radial decoration
- Round/soft-edged cards with organic border-radii
- Alternating layout with flowing section transitions
- Leaf and vine decorative accents
- Trellis/garden grid patterns as subtle backgrounds

## Typography
"Nunito" for headings — rounded, friendly, approachable. "Quicksand" for body — soft, clean, modern. Both sans-serif faces feel warm and human.

## Color Palette
Sky blue (#87ceeb), leaf green (#4caf50), and sunlight yellow (#ffd54f) create a bright optimistic palette. Warm white (#f5faf0) keeps things airy. Deep green (#1b5e20) and coral (#ff8a65) add grounding and warmth.

## Decorative Elements
- Sunburst radial gradients in the hero
- Organic flowing shapes via custom border-radius
- Leaf/vine decorations using CSS pseudo-elements and borders
- Warm sunlight gradient overlays
- Rounded soft cards with green accents
- Community garden trellis patterns

## Lightbox
Soft green-accented frame, bright overlay, sunlight-gold controls, organic rounded corners.

## Accessibility Note
High contrast throughout with bright warm colors. All animations respect prefers-reduced-motion.

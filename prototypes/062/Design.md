# Design Document — Prototype 062

## Concept
Paleontological museum exhibit. The portfolio presents projects as artifacts in a natural history collection. Earth strata, fossil textures, specimen labels.

## Color Palette
- Background: #F5F0E8 (parchment), #2C1810 (deep earth)
- Text: #2C1810 (primary), #7A6B5D (secondary)
- Accents: #C47E2E (amber), #8B7D71 (stone), #A6C48A (moss)
- Section backgrounds alternate between strata colors

## Typography
- "Playfair Display" (Google Fonts) for headings — scholarly serif
- "DM Mono" (Google Fonts) for labels, counters — typewriter feel
- System sans-serif for body
- Section labels styled as museum specimen tags (boxed, rotated)

## Layout
- Horizontal strata bands as section backgrounds
- Project cards framed like display cases (border with screws/rivet dots)
- Fossil crack SVG patterns as section dividers
- Specimen label layout: number, name, date, description in grid
- Footer styled as exhibit credit plaque
- Hero with earth cutaway cross-section visual

## Key Interactions
- Fossil-like crack pattern reveal on scroll
- Project card hover: specimen drawer pull-out effect
- Lightbox with brass frame border
- Strata lines animate on scroll

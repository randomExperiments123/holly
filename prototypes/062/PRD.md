# Product Requirements Document — Prototype 062

## Overview
Paleontological/natural history museum-inspired portfolio. Fossil textures, earth strata layers, amber and stone colors. Every section feels like a museum exhibit display case.

## Design Direction
- Inspiration: Natural History Museum exhibits, fossil displays, geological strata
- Earth strata horizontal banding (section dividers)
- Fossil-like textures via CSS (cracked patterns, sedimentary layers)
- Amber (#C47E2E), Stone (#8B7D71), Fossil (#D4C5B0), Deep Earth (#2C1810)
- Display case framing for project cards
- Specimen label typography

## Content
- 5 design projects with the standard project data
- About section (curator bio style)
- Contact section
- Download portfolio button

## Technical Requirements
- Pure HTML + CSS, zero JS dependencies
- Inline lightbox with minimal inline JS
- AVIF images
- Responsive mobile-first
- Accessible
- SEO-optimized with Schema.org

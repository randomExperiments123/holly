# Design Document — Prototype 037

## Concept
Modernist Grid applies rigorous grid theory to the portfolio. The grid itself is a visible design element — heavy lines, asymmetric placements, and a signature red accent.

## Color Palette
- Background: #f5f5f0 (warm off-white)
- Grid lines: #d4d4cc (subtle), #2c3e50 (bold)
- Accent: #c0392b (Swiss red)
- Text: #2c3e50 (dark slate)
- Light text: #7f8c8d

## Typography
- Inter (all weights) — clean, geometric sans-serif
- Heading weights: 700 (bold), Body: 400
- Monospaced numbers for project numbering

## Layout
- Strict 12-column invisible grid with visible heavy border lines
- Asymmetric card sizing (some 2-col, some 1-col on desktop)
- Grid lines extend beyond content areas
- Red accent line on hero
- Offset section titles

## Key Visual Elements
- Visible grid lines (CSS borders on columns)
- Asymmetric layout with intentional imbalance
- Red accent strips and underlines
- Monospaced project numbers in red
- Clean, precise spacing

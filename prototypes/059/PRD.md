# Prototype 059 — Chicano

## Overview
A portfolio page inspired by Chicano / lowrider aesthetic. Bold script lettering, ornate decorative borders, rich jewel tones, and intricate pattern details.

## Requirements
- All 5 projects with image slideshow
- Lightbox for full-screen image viewing
- Mobile-first responsive layout
- Accessible (ARIA, skip link, semantic HTML)
- SEO meta tags + Schema.org markup
- Pure HTML + CSS, inline lightbox JS only

## Design Direction
- Bold script / calligraphy typography
- Ornate decorative borders (CSS filigree patterns)
- Rich colour palette (deep red, gold, black, cream)
- Lowrider aesthetic elements (flame motifs, pinstripe details)
- Intricate ornamental frames and dividers

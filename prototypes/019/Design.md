# Design Document — Prototype 019

## Concept
Pop art meets graphic design portfolio. Bold, loud, and playful — inspired by comic books and pop artists like Roy Lichtenstein. Halftone dot backgrounds, primary colors, and comic-style borders create an energetic, fun experience.

## Color Palette
- Red: #ff0000
- Blue: #0039a6
- Yellow: #ffd700
- Black: #000000
- White: #ffffff
- Halftone patterns using CSS radial-gradient

## Typography
- Display: 'Bangers', cursive (comic book style)
- Body: 'Roboto', sans-serif (clean contrast)
- Headings use Bangers with letter-spacing
- Uppercase navigation, bold and impactful

## Layout
- Full-width yellow hero with black comic-style border
- Two-column grid with offset cards
- Comic-style speech bubble elements
- Halftone dot background patterns
- Bold black outlines on cards and buttons
- Playful hover effects (slight rotation, scale)

## Key Interactions
- Card hover: scale + slight rotation (playful)
- Button hover: bold color swap
- Inline lightbox with comic-style borders
- Halftone dot background animation on hover

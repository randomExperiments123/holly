# Design Document — Prototype 065

## Concept
Luxury marble. The portfolio emulates a high-end brand's lookbook printed on premium marble-textured stock. Subtle veining, gold foil accents, timeless elegance.

## Color Palette
- Background: #FAFAFA (Carrara white), #F5F3EF (warm marble)
- Text: #2C2C2C (charcoal), #8A8A8A (warm gray)
- Accents: #D4AF37 (gold), #B76E79 (rose gold), #E8E0D8 (beige)
- Marble veining: repeating linear-gradient patterns
- Gold gradient for metallic sheen

## Typography
- "Cormorant Garamond" (Google Fonts) — elegant serif for headings
- "Playfair Display" (Google Fonts) for display text
- "Montserrat" (Google Fonts) for body — clean sans-serif contrast
- Small caps for section labels
- Gold-colored drop caps on key paragraphs

## Layout
- Hero with full-width marble texture background and gold divider line
- Project cards as gallery frames (double border with gold inner line)
- Asymmetric grid layout (staggered masonry-like)
- Veining patterns as section dividers
- Footer as engraved marble plaque
- Generous white space with gold rule lines

## Key Interactions
- Subtle veining parallax on scroll
- Project card hover: gold foil shimmer
- Lightbox with gilded frame border
- Marble texture intensity shift on scroll
- Elegant fade transitions

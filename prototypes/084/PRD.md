# PRD: 084 — Asymmetrical Grid

## Overview
A deliberately unbalanced grid system. Swiss-grid principles but broken — overlapping elements, off-grid placement, intentional tension. The portfolio feels dynamic and unpredictable while remaining functional.

## Requirements
- Deliberately asymmetrical layout — elements float off-grid
- Overlapping content blocks and images
- Broken Swiss-grid aesthetic with tilted/skewed elements
- Dynamic visual tension in every section
- Project cards at varying angles and sizes
- Standard lightbox for project images
- Mobile-first, responsive, accessible

## Content Structure
1. Hero — Off-center, large + small type overlapping
2. Intro — Floating text block, not aligned to any grid
3. Projects — cards at varying positions, some overlapping
4. About — Asymmetrical two-column
5. Contact — Large block off to one side
6. Footer

## Constraints
- Pure HTML+CSS+inline JS
- AVIF images only
- No external dependencies
- Must pass accessibility checks

# Design Document — Prototype 032

## Concept
Industrial — the portfolio resembles a factory blueprint crossed with a warehouse aesthetic. Heavy, grounded, and utilitarian with metallic accents.

## Color Palette
- Background: #2c2c2c (concrete gray)
- Accent: #ff6b35 (industrial orange/rust), #8b7355 (weathered brass)
- Text: #d4d4d4 (light steel), #999 (worn gray)
- Card: #3a3a3a with metallic border gradients

## Typography
- 'Bebas Neue' (headings) — bold, condensed, industrial
- 'Roboto Condensed' (body) — readable, compact
- Uppercase headings throughout

## Layout
- Sturdy grid with heavy borders
- Rivet/bolt decorative elements via CSS
- Warning-striped accents (yellow/black)
- Fixed heavy-duty nav bar

## Key Interactions
- Metallic sheen hover effects
- Heavy box-shadow on hover (like lifting steel plates)
- Lightbox with keyboard navigation
- Rough edge transitions

# Design Document — Prototype 017

## Concept
Dark Academia meets graphic design portfolio. Rich, warm, scholarly — like a vintage leather-bound book. The design evokes old libraries, candlelight, and parchment.

## Color Palette
- Background: #f4ecd8 (aged paper)
- Text: #3b2f2a (ink brown)
- Accent: #8b4513 (rust)
- Marble/paper texture via CSS layered radial gradients

## Typography
- Body: 'Libre Baskerville', serif (scholarly, classic)
- Navigation: 'Source Sans Pro', sans-serif (clean contrast)
- Headings: 'Libre Baskerville', bold italic
- 18px base for comfortable reading

## Layout
- Single column for scholarly reading feel
- Max-width 800px article-style container
- Ornamental divider (❧) between sections
- Decorative border patterns on cards
- Vintage-inspired fixed header with paper texture

## Key Interactions
- Card hover: subtle sepia overlay
- Inline lightbox for project images
- Smooth scroll navigation
- Decorative hover effects on links

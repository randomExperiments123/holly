# Prototype 060 — Ghibli-inspired

## Overview
A portfolio page inspired by Studio Ghibli's aesthetic. Whimsical, hand-drawn feeling, soft watercolour backgrounds, warm nature tones, and charming organic details.

## Requirements
- All 5 projects with image slideshow
- Lightbox for full-screen image viewing
- Mobile-first responsive layout
- Accessible (ARIA, skip link, semantic HTML)
- SEO meta tags + Schema.org markup
- Pure HTML + CSS, inline lightbox JS only

## Design Direction
- Soft watercolour-gradient backgrounds
- Warm earthy nature tones (forest green, warm brown, sky blue, cream)
- Hand-drawn / organic feel through CSS shapes and borders
- Whimsical decorative elements (leaves, clouds, natural motifs)
- Cozy, storybook atmosphere

# Product Requirements Document — Prototype 027

## Overview
Glitch / Deconstructed portfolio for Holly Matthews. Broken grid layouts, skewed sections, overlapping elements, and CSS glitch animations with cyberpunk aesthetic.

## Design Direction
- Inspiration: Glitch art, deconstructivism, cyberpunk, broken layouts
- Dark (#0d0d0d) with cyan (#00f0ff) and pink (#ff0066)
- CSS glitch animation on headings
- Skewed, overlapping sections
- IBM Plex Mono font for technical feel
- Chaotic yet controlled

## Content
- 5 design projects: Nordica Coffee, Urban Perspectives, Motion & Interaction, Botanic, Dreamscapes
- About section
- Contact section
- Download portfolio button

## Technical Requirements
- Pure HTML + CSS, zero JS dependencies
- Inline lightbox with minimal inline JS
- AVIF images
- Responsive mobile-first
- Accessible
- SEO-optimized with Schema.org

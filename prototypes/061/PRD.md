# Product Requirements Document — Prototype 061

## Overview
French New Wave cinema-inspired portfolio. Bold title cards, letterboxed imagery, high contrast black & white with a single accent color, grainy film textures, and dramatic typography evoking 1960s French cinema.

## Design Direction
- Inspiration: Jean-Luc Godard, François Truffaut film posters and title sequences
- Cinematic letterboxing (2.35:1 aspect ratio) on hero and project images
- High contrast B&W photography with pops of red (#E8202F)
- Grainy film texture overlay via CSS pseudo-elements
- Bold, centered title cards with expressive typography
- Split-screen and jump-cut inspired layout transitions

## Content
- 5 design projects with the standard project data
- About section with cinematic bio treatment
- Contact section
- Download portfolio button

## Technical Requirements
- Pure HTML + CSS, zero JS dependencies
- Inline lightbox with minimal inline JS
- AVIF images
- Responsive mobile-first
- Accessible
- SEO-optimized with Schema.org

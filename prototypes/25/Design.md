# Design Document — Prototype 025

## Concept
The infinite cosmos as a design language. A deep space background dotted with twinkling stars and floating particles. Nebula-inspired gradients in purple and pink create celestial glow around content cards. The portfolio becomes a journey through the stars.

## Color Palette
- Background: #05051a (deep space black-blue)
- Secondary bg: #0a0a2e (slightly lighter space)
- Nebula purple: #6c63ff
- Cosmic pink: #ff6584
- Star white: #ffffff
- Nebula accent: #b388ff (light purple)
- Text: #ffffff, #b0b0d0 (muted star light)

## Typography
- Headings: 'Orbitron', sans-serif (futuristic, space-age)
- Body: 'Rajdhani', sans-serif (clean, readable)
- Orbitron's geometric, futuristic feel matches space theme
- Rajdhani provides excellent readability

## Layout
- Full dark space background
- CSS star particles across the page
- Centered max-width container (1100px)
- Two-column grid on desktop, single on mobile
- Transparent fixed header with blur
- Cards with subtle glow borders (purple/pink)
- Section backgrounds with nebula gradient overlays
- Rounded corners (12px)

## Key Interactions
- Stars twinkle via CSS animation
- Card glow intensifies on hover
- Nebula gradient shift animation
- Inline lightbox for project images
- Smooth scroll navigation

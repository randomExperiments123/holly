# PRD: Coastal Aesthetic Portfolio

## Purpose
Present Holly Matthews' portfolio with a relaxed coastal/beach theme. Ocean blues, sandy tans, and seafoam greens with wave-inspired decorative elements.

## Requirements
- Beach colour palette: sand (#F4E4C1), ocean blue (#0077B6), seafoam (#98F5E1), driftwood (#8B7355)
- Relaxed sans-serif typography (Nunito, Open Sans)
- Wave motif decorative elements via CSS
- Coastal imagery aesthetic
- Light, airy, spacious layout
- Driftwood-style textures via CSS gradients
- Lightbox with ocean-blue treatment

## Content
Same 5 projects with AVIF images.

## Constraints
- Pure HTML+CSS, inline JS only
- Mobile-first responsive
- Accessible (ARIA, skip link, semantic HTML)
- AVIF images only
- Schema.org markup
- No external dependencies

# Design Document — Prototype 033

## Concept
Romantic — the portfolio is designed like a love letter. Soft gradients, floating heart decorations, and elegant script typography create a dreamy, heartfelt atmosphere.

## Color Palette
- Background: #fce4ec (pale pink)
- Primary: #e91e63 (hot pink)
- Dark: #880e4f (dark rose)
- Accent: #f8bbd0 (light pink)
- Gold: #d4af37 (warm accent)

## Typography
- Playfair Display italic + cursive (Google Fonts)
- Smooth, flowing letterforms
- Ornamental ampersands and flourishes via CSS

## Layout
- Hero with heart pattern overlay (CSS pseudo-elements)
- Cards with soft rounded corners and pink gradients
- Decorative curves and flourishes
- Centered, symmetrical layout for elegance

## Key Interactions
- Floating heart animation in hero
- Soft glow hover effects
- Gentle gradient transitions
- Lightbox with keyboard navigation

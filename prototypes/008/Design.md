# Design Document — Prototype 008

## Concept
Retro vaporwave — a nostalgic trip to the neon-lit 80s and 90s. Bold gradients, chrome text, grid backgrounds, and retro computing aesthetics.

## Color Palette
- Background: #0a0a2e (deep night)
- Neon Pink: #ff6ec7
- Neon Blue: #00d4ff
- Neon Purple: #b026ff
- Gold: #ffd700
- Text: #ffffff, rgba(255,255,255,.7)

## Typography
- 'Orbitron' (Google Fonts) — futuristic, geometric
- 'Rajdhani' — clean, tech-inspired
- Chrome/textured text effects via CSS
- All-caps for impact

## Layout
- Centered hero with retro grid
- Horizontal scrolling project shelf
- Neon glow effects on all interactive elements
- CRT scanline overlay
- Sunset gradient backgrounds
- Grid perspective lines (retro style)

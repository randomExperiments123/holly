# PRD: Acid Grunge (069)

## Purpose
Present Holly Matthews' portfolio with a late-90s rave/psychedelic aesthetic. Chaotic layouts, neon colours on black, warped typography, grunge textures, and a general "underground club flyer" energy.

## Requirements
- Black background with neon green (#00ff00), magenta (#ff00ff), cyan (#00ffff), yellow (#ffff00)
- CSS transform-based text distortion (skew, rotate, scale on headings)
- Chaotic, asymmetrical layouts with elements breaking grid
- Grunge/noise texture overlay
- Glitch-inspired decorative effects
- Maintain all 5 project cards, lightbox, skip link, SEO, Schema.org, ARIA
- Mobile-first responsive layout

## Constraints
- Pure HTML + CSS, zero JS dependencies (except inline lightbox)
- AVIF images only
- Self-contained
- No external fonts

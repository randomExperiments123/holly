# PRD: 085 — Monospace Only

## Overview
A strict monospace-only portfolio. One typeface throughout. Typewriter aesthetic. Strict grid. No decorative images — pure typographic hierarchy and whitespace create the visual interest.

## Requirements
- Single monospace font throughout (Courier New / system monospace)
- Strict grid alignment — everything snaps to grid
- No images used as decoration (project images still shown in lightbox)
- Typewriter feel — ASCII-style dividers, underscores
- Pure typographic hierarchy — size, weight, spacing only
- All text content visible, nothing hidden behind images
- Standard lightbox for project images
- Mobile-first, responsive, accessible

## Content Structure
1. Hero — Typewriter-style header with underscores and pipes
2. Projects — ASCII-boxed project cards
3. About — Plain text with typewriter formatting
4. Contact — Plain email
5. Footer

## Constraints
- Pure HTML+CSS+inline JS
- AVIF images only (only in lightbox)
- No external dependencies
- Must pass accessibility checks

# Prototype 055 — Bauhaus Vol. 3: Design Decisions

## Color Palette
- `#f5f5f5` (white) — clean background
- `#1a1a1a` (black) — text, structure
- `#e30613` (red) — bold accent
- `#0039a6` (blue) — secondary accent
- `#ffd700` (yellow) — highlight

## Typography
- Geometric sans-serif font-family stack
- Heavy (700-900) weights for headings
- Functional, no-nonsense spacing

## Layout
- Asymmetrical hero with overlapping circle/square/triangle
- Project grid: staggered columns (different heights)
- Thick underlines and borders (functional)
- No rounded corners — strict 90-degree angles

## Rationale
Different from 009 (split-column hero, red-centric) and 045 (dark theme, gold accent, ornate badges). This version is pure functional: asymmetrical grid, overlapping primary-color shapes, strict right angles. The layout feels more like a Bauhaus poster — bold, unbalanced yet perfectly weighted.

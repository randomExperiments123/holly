# Design Document — Prototype 026

## Concept
Typography is the hero. Massive 200px headings dominate the viewport, making the text itself the primary visual experience. The design strips away everything unnecessary — just black, white, and a single punch of #ff3366. This is typographic minimalism at its most extreme.

## Color Palette
- Background: #ffffff (white)
- Text: #000000 (black)
- Secondary bg: #000000 (black sections)
- Accent: #ff3366 (vibrant pink-red)
- Light text: #666666
- Card backgrounds: #f5f5f5

## Typography
- All text: 'Inter', sans-serif
- Extreme size hierarchy: 200px hero, 96px section titles, 24px body
- Weight variations: 900 (black) for hero, 700 for headings, 300 for body
- Tight letter-spacing for display, generous for body

## Layout
- Clean white background
- Massive overlapping typography
- Centered max-width container (900px — narrower for focus)
- Single column project layout (hero treatment)
- Full-height hero section with 200px name
- Cards minimal — just images and type
- Generous whitespace (120px+ sections)
- Grid lines optional (subtle)

## Key Interactions
- Accent color appears on hover
- Clean, no-fuss transitions
- Inline lightbox for project images
- Smooth scroll navigation

# Design Document — Prototype 042

## Concept
Maximalist rejects minimalism entirely. Layered patterns, clashing colors, multiple typefaces, and dense compositions create a rich, overwhelming visual experience that rewards exploration.

## Color Palette
- Hot pink: #ff0066
- Neon green: #00ff66
- Electric blue: #0066ff
- Bright orange: #ff6600
- Yellow: #ffcc00
- Deep purple: #6600ff
- No neutral colors

## Typography
- Multiple fonts: Playfair Display (headings), Inter (body), monospace (accents), Impact (banners)
- Mixed sizes and weights
- Intentional font clashes

## Layout
- Dense, layered grid
- Overlapping cards and text
- Pattern-on-pattern backgrounds
- Busy borders and frames
- No whitespace — every pixel used

## Key Visual Elements
- Multiple overlapping CSS background patterns
- Clashing color combinations
- Decorative borders everywhere
- Layered text (text on top of text)
- Pattern mixing (stripes + dots + zigzags)
- Visual noise as intentional design

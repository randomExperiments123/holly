# Design Document — Prototype 040

## Concept
Abstract Expressionist turns the portfolio into a canvas. Bold paint splashes, chaotic layouts, and dripping color borders create an unpredictable, energetic feel.

## Color Palette
- Canvas white: #ffffff (background)
- Paint red: #ff4444
- Paint blue: #4488ff
- Paint yellow: #ffcc00
- Paint green: #44ff88
- Ink black: #1a1a1a (text)

## Typography
- Inter (sans-serif, clean contrast to chaos)
- Bold weights for impact
- Irregular text alignment

## Layout
- Asymmetric, seemingly random card placement
- Cards at slight rotations
- Paint-splatter background elements
- Broken grid — items overflow edges
- Drip effects on section borders

## Key Visual Elements
- Paint splatter background via CSS pseudo-elements
- Paint-drip borders (asymmetric border shapes)
- Rotated cards for chaotic feel
- Color-block section dividers
- Overlapping elements
- Drip animation effects

# PRD: Western / Cowboy Aesthetic Portfolio

## Purpose
Present Holly Matthews' portfolio with a wild west/cowboy theme. Leather browns, turquoise accents, and wanted-poster styling.

## Requirements
- Western palette: leather brown (#8B4513), turquoise (#40E0D0), cream (#FFF8DC), burnt orange (#CC5500)
- Rodeo/western typography
- Distressed/dirt-textured backgrounds via CSS
- Wanted-poster decorative borders (dashed, decorative)
- Star badges as decorative elements
- Saloon-style wood textures
- Lightbox with wanted-poster treatment

## Content
Same 5 projects with AVIF images.

## Constraints
- Pure HTML+CSS, inline JS only
- Mobile-first responsive
- Accessible (ARIA, skip link, semantic HTML)
- AVIF images only
- Schema.org markup
- No external dependencies

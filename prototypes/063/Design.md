# Design Document — Prototype 063

## Concept
Geode/Crystal aesthetic. The portfolio is built like a crystal geode — rough dark exterior, brilliant jewel-toned interior. Sharp facets, angular cuts, geometric precision.

## Color Palette
- Background: #0D0D12 (deep cavern), #1A1A24 (dark violet)
- Text: #E8E8F0 (crystal white), #8888AA (dimmed)
- Accents: #9B59B6 (Amethyst), #2ECC71 (Emerald), #3498DB (Sapphire), #F1C40F (Citrine)
- Gradient faceted surfaces

## Typography
- "Space Grotesk" (Google Fonts) — geometric, modern, angular
- "Orbitron" (Google Fonts) for numbers — futuristic
- Monospace for counters
- All caps headings with sharp letter-spacing

## Layout
- Hero with faceted diamond clip-path mask
- Project cards as irregular hexagons (clip-path polygon)
- Grid broken into angular shards
- Crystal lattice decorative elements (CSS-drawn)
- Footer as crystal base/pedestal
- Section transitions with chevron/Z-shaped dividers

## Key Interactions
- Faceted edge hover: crystal rotation/sheen effect
- Lightbox with faceted diamond border
- Sparkle/pulse animation on accent elements
- Geometric pattern background shifts on scroll

# Product Requirements Document — Prototype 063

## Concept
Geode/Crystal-inspired portfolio. Sharp faceted edges, geometric crystal shapes, amethyst/emerald/sapphire jewel tones on dark backgrounds. Every element cut like a gemstone.

## Design Direction
- Inspiration: Geode formations, crystal structures, faceted gemstones
- Sharp angular geometry — hexagons, diamonds, chevrons
- Dark background (#0D0D12) with vibrant jewel-tone accents
- Amethyst (#9B59B6), Emerald (#2ECC71), Sapphire (#3498DB)
- Faceted edges (clip-path polygons) on cards and sections
- Crystalline sparkle effects via CSS gradients

## Content
- 5 design projects with the standard project data
- About section
- Contact section
- Download portfolio button

## Technical Requirements
- Pure HTML + CSS, zero JS dependencies
- Inline lightbox with minimal inline JS
- AVIF images
- Responsive mobile-first
- Accessible
- SEO-optimized with Schema.org

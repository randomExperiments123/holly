# PRD: 083 — Split Complementary

## Overview
A strict split-complementary color scheme portfolio. Three hues (base + two complements) define every visual decision. Color-blocked sections create a harmonious clash — minimalist but bold.

## Requirements
- Strict split-complementary palette only (no neutrals except white/off-white)
- Color-blocked sections — each section is a solid color
- Minimalist layout — content sparse, lots of breathing room
- Harmonious clash — colors that work together despite tension
- Project cards are simple colored blocks with text + thumbnail
- Standard lightbox for project images
- Mobile-first, responsive, accessible

## Content Structure
1. Hero — Split-color background, centered type
2. Projects — alternating background colors
3. About — one solid color with white text
4. Contact — final color block
5. Footer

## Constraints
- Pure HTML+CSS+inline JS
- AVIF images only
- No external dependencies
- Must pass accessibility checks

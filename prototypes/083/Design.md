# Design Document: 083 — Split Complementary

## Concept
Pure color theory in practice. A split-complementary triad drives every visual choice. The result is vibrant, harmonious, and intentionally constrained — proving that limitation breeds creativity.

## Color Palette (Split-Complementary Triad)
- Base: #264653 (deep teal)
- Complement A: #e76f51 (burnt orange)
- Complement B: #e9c46a (golden yellow)
- Text on dark: #ffffff
- Text on light: #1a1a1a

## Typography
- Headings: "Segoe UI", system sans-serif, bold, uppercase
- Body: system sans-serif, regular weight
- Generous letter-spacing on headings

## Interactions
- Subtle color shift on project card hover (brightness increase)
- Section transitions: hard cuts between color blocks (no fade)
- Lightbox: minimal, matches current section color
- Links: underline only on hover

## Layout
- Full-viewport-height sections each in a solid color
- Content centered both axes per section
- Project cards: 2-column grid on desktop, 1 on mobile
- Cards have thin white border, transparent background

## Key Design Decisions
- No neutrals except white — every pixel is a palette color
- Hard section transitions for bold architectural feel
- Typography is secondary to color — it steps back
- Images sit inside colored cards without background

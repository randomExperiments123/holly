# PRD: Wabi-Sabi — 089

## Purpose
Embrace imperfection, asymmetry, and natural materials. Inspired by the Japanese wabi-sabi aesthetic — weathered textures, muted earth tones, minimalist composition with deliberate irregularities.

## Requirements
- Muted earth tone palette: clay, ochre, sage, stone
- Asymmetric grid layout (no equal columns)
- Weathered/broken texture via CSS layered gradients
- Irregular borders (hand-drawn feel via dashed/outset combinations)
- Minimalist with generous whitespace
- Natural, organic feel throughout
- Schema.org markup on projects
- Inline lightbox with keyboard nav

## Constraints
- Pure HTML + CSS + inline JS
- No photography or external textures
- All texture must be CSS-native

## Success Criteria
- Feels handcrafted and organic
- Asymmetry reads as intentional
- Color palette evokes earth and stone
- Images sit naturally within the weathered framework
- Texture adds depth without overwhelming

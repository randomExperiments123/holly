# Prototype 056 — Origami

## Overview
A portfolio page designed with an origami / folded paper aesthetic. Clean geometric folds, subtle layered shadows, pastel colour palette with plenty of white space.

## Requirements
- All 5 projects displayed with image slideshow
- Lightbox for full-screen image viewing
- Mobile-first responsive layout
- Accessible (ARIA, skip link, semantic HTML)
- SEO meta tags + Schema.org markup
- Pure HTML + CSS, inline lightbox JS only

## Design Direction
- Folded paper aesthetic using CSS transforms and layered box-shadows
- Pastel colour palette (soft pink, mint, baby blue, lavender) + white
- Clean geometric shapes with subtle angular clip-paths
- Card-style project tiles with paper-fold corner effects
- Light, airy feel with soft shadows

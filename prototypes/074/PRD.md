# PRD: Safari Adventure Aesthetic Portfolio

## Purpose
Present Holly Matthews' portfolio with a bold safari/adventure theme. Khaki, olive, and rust palette with animal-print decorative elements.

## Requirements
- Safari palette: khaki (#C3B091), olive (#556B2F), rust (#B7410E), cream (#F5E6D3)
- Bold, adventurous typography
- Animal-print decorative patterns via CSS
- Adventure-themed section headings
- Rugged, exploration feel
- Lightbox with safari-treatment styling

## Content
Same 5 projects with AVIF images.

## Constraints
- Pure HTML+CSS, inline JS only
- Mobile-first responsive
- Accessible (ARIA, skip link, semantic HTML)
- AVIF images only
- Schema.org markup
- No external dependencies

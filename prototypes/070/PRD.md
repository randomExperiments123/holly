# PRD: IndieWeb (070)

## Purpose
Showcase Holly Matthews' portfolio in a nostalgic, hand-coded 90s personal website aesthetic. Inspired by the indie web movement, GeoCities, and early personal homepages — monospace, minimal, with "guestbook" energy.

## Requirements
- Monospace-only typography (Courier, monospace)
- Minimalist "hand-coded HTML" feel with visible structural elements
- 90s web-safe colour palette (#000000, #00ff00, #c0c0c0, #ffffff, #808080, #000080)
- ASCII-style borders and separators (hyphens, pipes, plus signs simulated via CSS)
- Under construction / guestbook visual motifs
- Simple, uncluttered layout with maximum readability
- Maintain all 5 project cards, lightbox, skip link, SEO, Schema.org, ARIA
- Mobile-first responsive layout

## Constraints
- Pure HTML + CSS, zero JS dependencies (except inline lightbox)
- AVIF images only
- Self-contained
- No external fonts

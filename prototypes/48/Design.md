# Design Decisions — Prototype 048

## Concept
Geometric tribal patterns inspired by Indigenous weaving and textile traditions. The design uses diamonds, chevrons, and stepped geometric motifs to create borders and decorative elements. The approach is respectful — drawing from geometric pattern traditions rather than specific sacred symbols.

## Layout
- Full-width hero with diamond patterned background
- Project cards with tribal-inspired border frames
- Chevron section dividers
- Patterned footer

## Typography
Inter provides clean, neutral readability that lets the pattern work shine without competing.

## Color Palette
Earthy tones of deep brown, terracotta red, cream, and forest green create a grounded, natural feel rooted in the landscape.

## Pattern Elements
- Diamond motifs created with CSS transforms
- Chevron borders using repeating linear gradients
- Stepped geometric patterns with box-shadow
- Triangle motifs via border techniques

## Lightbox
Dark overlay with geometric tribal frame pattern and earth-toned controls.

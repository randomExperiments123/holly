# Holly Matthews — Portfolio

A lightweight, responsive portfolio website showcasing 5 design projects.

## Features
- **Pure HTML, CSS, zero JS dependencies** — minimal, fast, maintainable
- **AVIF images** — best compression/quality ratio
- **SEO-optimized** — semantic HTML, meta tags, Schema.org markup
- **Accessible** — skip link, ARIA labels, reduced-motion support, proper contrast
- **Responsive** — CSS Grid with mobile-first breakpoints

## Local Preview

```bash
# Option 1: Python
python3 -m http.server 8000

# Option 2: Node
npx serve .

# Then open http://localhost:8000
```

## Download

Clone or download the repo:

```bash
git clone <repo-url>
```

Or download as ZIP from GitHub.
